# Rest Countries

A web application that displays information abount countries using the REST Countries API. Users can search for countries and view details like population, region, capital, and flag.

## Live Demo
https://rest-countries-keshav.netlify.app

## Features
- View all countries
- Search countries by name
- See country details (population, region, capital, flag)
- Responsive design

## Technologies Used
HTML
CSS
JavaScript
REST Countries API
